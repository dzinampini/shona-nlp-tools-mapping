Shona NLP Ecosystem

Mapping of Shona Language Processing Tools and Datasets

Research on Shona Natural Language Processing is currently scattered across journal papers, conference proceedings, theses and software repositories. This project brings those resources together into a single searchable catalogue. By making existing tools and datasets visible, researchers can identify gaps, avoid duplication of effort, encourage collaboration and focus future work on areas that remain underdeveloped.

Contributing

This project is community-driven. Whether you're a researcher, linguist, software developer, or simply aware of Shona NLP resources, your contribution helps build a comprehensive, openly accessible ecosystem map.

Why Contribute?

Many Shona NLP tools and datasets remain scattered across research papers, theses and repositories. By contributing, you help improve discoverability, reduce duplicated effort and strengthen future research.

Developers
  1. Fork the repository.
  2. Create a new branch.
  3. Update data.json.
  4. Add supporting references.
  5. Submit a Pull Request.
