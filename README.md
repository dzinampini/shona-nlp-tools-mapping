# Shona NLP Tools Mapping
An open repository for mapping, documenting, and tracking the development of Natural Language Processing (NLP) tools, resources, and datasets for the Shona language.

This repository aims to provide a centralised overview of the current state of Shona NLP by cataloguing existing tools, models, datasets, libraries, applications, and research contributions. It will support researchers, developers, students, and organisations working on low-resource African language technologies by identifying available resources, existing gaps, and opportunities for collaboration and innovation.
